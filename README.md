# Shortener

Сервис для создания сокращенных ссылок. Проект создан в учебных целях для практики DevOps навыков, в частности: 
- Docker
- Docker Compose  
- Развёртывания приложений на Linux-серверах

## Возможности

- Создание коротких ссылок
- Хранение созданных ссылок

## Технологии

- FastAPI
- VueJs
- PostgreSQL
- Redis
- Docker
- Docker Compose
- Nginx

## Запуск

1. Переименовать (или скопировать) файл .env.example в .env и ввести туда свои данные
2. Выполнить команду:
```bash
docker compose up -d
```
